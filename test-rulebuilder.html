Instructions for Manual Testing

  3. Click on the conditional layout block to select it
  4. Check the settings panel on the right - you should see the RuleBuilder
  5. Try switching between Visual and Code modes
  6. Try adding rules and conditions
  7. If the app doesn't crash or freeze, the fix is successful! ✅
