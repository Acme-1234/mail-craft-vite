Testing Instructions

1. Add a Button Block

Add a button block to the canvas and select it to open the settings panel. You should see enhanced size inputs for padding and border radius.

2. Test Button Padding

Test the new padding control:

  • Use the slider to adjust padding values
  • Switch between Slider, Input, and Box tabs
  • Try the Box model view for individual sides
  • Test preset buttons (None, Small, Medium, Large)
  • Change units (px, em, rem, %)
  • Use the reset button to clear values

3. Test Border Radius

Test the border radius control with slider and presets (None, Small, Medium, Large, Round).

4. Test Container Styles

Add any block and test the Container Styles section:

  • Test padding with box model controls
  • Test margin with negative values support
  • Try the Auto preset for margin

5. Visual Feedback Test

Verify that size changes are immediately reflected in the canvas and that the controls provide visual feedback.
