Testing Instructions

1. Add a Button Block

Add a button block to the canvas and select it to open the settings panel. You should see enhanced color pickers for background and text colors.

2. Test Background Color Picker

Click on the Background Color field and test:

  • HEX format (e.g., #FF6B6B)
  • RGB format (e.g., rgb(255, 107, 107))
  • HSL format (e.g., hsl(0, 100%, 71%))
  • RGBA format (e.g., rgba(255, 107, 107, 0.8))
  • Common color palette selection
  • Native color picker
  • Eyedropper tool (if supported by browser)

3. Test Text Color Picker

Test the same features on the text color picker to ensure consistency across all color inputs.

4. Test Container Styles

Add any block and test the container styles color pickers for background and text colors in the Container Styles section.

5. Test Document Settings

Go to Document Settings and test the background color picker for the overall email background.

6. Visual Feedback Test

Verify that color changes are immediately reflected in the canvas and that the color preview in the picker shows the correct color.
